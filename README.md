# IRCMod
![](https://img.shields.io/github/downloads/ilovemegumin/IRCMod/total.svg)


### 解説
このModはシンプルなIRCModで違うサーバー間でチャットを可能にします。

### Build方法

- JDKをセットアップ(21)
- Gradle > Build を選択
- 完了

### GUIの開き方
Jボタンを押してください
